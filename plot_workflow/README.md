## Analysis workflow
The purpose of this code is to semi-automatically run analyses on the data produces by data_workflow. Key aspects include distributions of any desired properties, analyses of matching efficiencies of jets and subjets, especially on the effect of cone radius, decay channel and other event properties.

## How to run
Run all of the analysis with
```bash
./run.sh all --cores="" --config # optional flags
```
The source and paralellization are the same as in data_workflow. Note that this will require all files generated by data_workflow to be present and will make $\sim 30$ plots per dataset. After you have run the entire analysis, to rerun some parts, you can just delete the corresponding result (.png -file) and rerun the above command. To only run spesific parts of the analysis, refer to this:
| Rule | Analysis |
|----------|:-------------:|
| jet_plots | Plots a comparison of the distributions of jet properties specified in 'params' -flag | 
subjet_plots | Plots a comparison of the distributions of subjet properties from 'params' flag| 
| jet_effs | Calculates and compares the matching efficiencies of jets with respect to: Higgs $p_T$, pileup, $\tau$ asymmetry. Also compares energy response and $\Delta R(\tau, \text{jet})$. |
| subjet_effs | Calculates and compares subjet matching efficiencies (defined against all events and already matched parent jets) with both loose (>1 subjet) and tight (2 subjets) criteria. |
|effs_channel| Compares the effect of decay channel on matching efficiency for jet types and also plots the distribution of decay products and the effect of adding semileptonic channels to a dataset for reference. Fits a simple empirical model to these detailed in the slides.|
|clustering_kinematics | Computes observable $\min_{\text{jets}}\max\Big(\Delta R(\tau_1 , \text{jet}), \Delta R(\tau_2 , \text{jet}) \Big)$ and plots its distribution in different Higgs $p_T$ ranges for different jets, as well as scatter plots with respect to $\Delta R (\tau_1, \tau_2)$. This is to analyse clustering kinematics at $\Delta R_{\text{jet}}\sim \Delta R(\tau_1, \tau_2)$.

The analyses can be customized with these flags:
<center>

|Flag (set to preset value)|Usage
|-----------|:---------:|
|data="MADGRAPH, POWHEG"|Specify what data to process. Add paths to own datasets to DATA_DIRS dictionary in the top of Snakefile.
|channel="both"|Specify decay channels: "hadhad_only", "inclusive_only"
| exclude="" | Run process excluding some jet types, e.g. "AK4, Tau" |
|jet-params="X_mass, X_pt, X_mass/X_pt, abs(X_eta)"| Properties the distribution of which will be plotted in jet_plots. X to replace jet type in skim format.
|subjet-params="X_Subjet_mass, X_Subjet_pt, X_Subjet_mass/X_Subjet_pt, abs(X_Subjet_eta)"|Same as above but for the AK8 and AK15 subjet properties.

</center>


Note that the data for the analysis that you want to do has to be generated with the corresponding flags in data_workflow.
